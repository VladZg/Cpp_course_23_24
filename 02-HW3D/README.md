## How to build

At first clone the repo:
```bash
git clone <repo-url>
cd ./02-HW3D
```

Then to build a project run this in your terminal:
```bash
mkdir ./build
cd ./build
cmake ..
make
```

To run a program:
```bash
./main
```

To run tests:
```bash
./test_2D
./test_3D
./triangles
```
